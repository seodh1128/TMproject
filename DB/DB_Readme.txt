Oracle DataBase Export(백업) / import(불러오기)

필수 사항

CMD 창에서 수행 가능 - Oracle DB가 설치되어 있어야 함.

백업 방법

전체 DB 백업
exp system/password@xe full=y file=c:\해당 경로\파일명.dmp
특정 유저의 DB 백업
exp system/password@xe owner=유저명 file=c:\해당 경로\파일명.dmp

복원 방법

전체 DB 복원
imp system/password@xe file=file=c:\해당 경로\파일명.dmp
특정 유저의 DB 복원
exp system/password@xe fromuser=유저명 touser=유저명 file=c:\해당 경로\파일명.dmp

백업/복원 작업 수행시,  하단 기존 내역에 이어서 해당 내역(일자,작업자명,작업내용(간략히)을 적어주세요.
